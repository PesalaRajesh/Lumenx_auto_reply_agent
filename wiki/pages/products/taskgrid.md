# TaskGrid

## Overview

**TaskGrid** is a lightweight task tracker designed for teams that need spreadsheet simplicity with project management power. It combines the familiar spreadsheet grid interface with robust task organization features, making it ideal for ops teams, founders, and operations leads who find traditional project management tools too complex.

**Tagline:** Spreadsheet-fast task tracking.

## Key Features

- **Spreadsheet-style task grid** — Organize tasks in a familiar, intuitive layout
- **Custom fields and tags** — Tailor your workspace to your team's specific needs
- **Saved views per teammate** — Each team member can create and save personalized task views
- **CSV import and export** — Move data in and out of TaskGrid easily
- **Slack and email digests** — Stay updated without leaving your communication tools

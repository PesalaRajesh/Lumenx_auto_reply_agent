# DocuMerge

## Overview

DocuMerge is a browser-based PDF management tool that lets you merge, split, compress, and sign PDFs from a single clean interface. Whether you're combining 100 files into one tidy PDF or performing batch operations, DocuMerge handles the heavy lifting without requiring file uploads to external servers.

**Best for:** Anyone who works with PDFs more than once a week.

## Core Features

- **Merge & Split** — Combine multiple PDFs or extract specific pages
- **Rotate & Reorder** — Rearrange pages in any order
- **Compress** — Reduce file size without quality loss
- **Local Processing** — All operations run in your browser; documents stay private by default
- **Optional Cloud History** — Save your work history to the cloud if desired
- **Batch Rename** — Rename multiple files simultaneously
- **Digital Signatures** — Sign PDFs directly in the application

## Common Customer Questions

**Q: Does DocuMerge upload my PDFs to the cloud?**  
A: No. By default, all processing happens locally in your browser. Your documents never leave your device unless you explicitly choose to save them to cloud storage or enable cloud history.

**Q: Can I use DocuMerge offline?**  
A: Yes, the core PDF processing works offline. Cloud features (history, integrations) require an internet connection.

**Q: What's the difference between Pro and Team plans?**  
A: Pro is for individuals with unlimited daily processing. Team includes up to 10 seats, making it ideal for small teams sharing workflows.

**Q: Can I try DocuMerge before paying?**  
A: Yes! The Starter plan is free with a daily file limit, so you can test all features risk-free.
